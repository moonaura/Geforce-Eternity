# GeForce Eternity

**GeForce Eternity** is an unofficial, community-driven desktop client for NVIDIA GeForce NOW. Built with Electron, it brings a native desktop experience to Windows, macOS, and Linux, along with extras like Discord Rich Presence and other quality-of-life improvements.

## 🎮 Features

- 🧩 **Custom User Agent**: Switch between GFN Eternity, Chrome, and Firefox user agents for optimal compatibility.
- 🌐 **Multi-Language Support**: Choose your preferred GFN language directly from the sidebar.
- 🎯 **Auto Focus**: Automatically focuses the window when your gaming rig is ready.
- 🔇 **Auto Mute**: Automatically mutes audio when the window loses focus.
- 💬 **Discord Rich Presence**: Share your gaming status with real-time updates on Discord.
- 🛠️ **Sidebar Controls**: Quick access to **Reload GFN** and **Reset to Default** settings.
- 🛡️ **Idle Guard**: Reliable inactivity prevention for long-running streaming sessions.
- 🖥️ **Multi-Platform**: Native builds available for Windows, macOS, and Linux.

*And more features coming over time!*

## ⌨️ **Usage**

Press `Ctrl+I` to open sidebar to access GeForce Eternity features.

## 🧰 Build

### Prerequisites
You will need to install [npm](https://www.npmjs.com/), the Node.js package manager. On most distributions, the package is simply called `npm`.

1. Clone the repository:
    ```bash
    git clone https://github.com/moonaura/GeForce-Eternity.git
    cd GeForce-Eternity
    ```
2. Install dependencies:
    ```bash
    npm install
    ```
3. Start the app:
    ```bash
    npm start
    ```
